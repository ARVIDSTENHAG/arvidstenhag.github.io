Strava Runner Predictor – Fullstack Chrome Extension

An advanced Fullstack Chrome Extension (Manifest V3) engineered with a dedicated Node.js/Express backend to provide runners with precise, data-driven marathon predictions using real-time Strava API data.

The architecture features a secure OAuth 2.0 flow where sensitive client secrets are managed via server-side environment variables, ensuring maximum data integrity. By applying the Riegel formula to analyze historical performance, the application transforms raw training data into actionable, personalized race forecasts.
