Side projects

Strava Runner Predictor – Fullstack Performance Forecasting

A sophisticated Fullstack Chrome Extension (Manifest V3) integrated with a dedicated Node.js/Express backend, designed to bridge the gap between raw training data and actionable race goals. By leveraging the Strava API, the application fetches real-time performance metrics to deliver precise marathon time predictions directly in the browser.

The architecture prioritizes security and technical integrity, featuring a robust OAuth 2.0 implementation where sensitive credentials are encapsulated within server-side environment variables (.env) to protect user data. Utilizing the Riegel formula ($T_2 = T_1 \times (D_2 / D_1)^{1.06}$), the extension performs advanced calculations on historical activity data to provide athletes with reliable, personalized race-day projections.
